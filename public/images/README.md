# Images

Store all app images in this folder.

- Hero image: `hero/hero-main.*`
- Legacy images: `legacy/owner.*`, `legacy/children.*`
- Gallery images: `gallery/gallery-01.*`, `gallery/gallery-02.*`, ...

To add or replace gallery photos:

1. Put image files into `public/images/gallery/`.
2. Update the list in `src/data/galleryImages.ts`.

Use same aspect ratio for gallery photos for best layout (recommended 4:3).
